# Titã - Plataforma de Atendimento Inteligente

Solução personalizada de atendimento multicanal projetada para Provedores de Internet (ISP). Esta plataforma evolui o atendimento da empresa, substituindo ferramentas legadas por uma interface premium e IA orquestradora.

## 🚀 Funcionalidades

- **Atendimento Multicanal**: Integração com WhatsApp, Instagram e WebChat.
- **IA Orquestradora**: Camadas de RAG e Fine-tuning para respostas precisas baseadas na base de conhecimento da empresa.
- **Integrações Nativas**: Conexão direta com HubSoft (CRM/ERP), Discord (Notificações) e ASGP (Ocorrências).
- **Interface Premium**: Design moderno, responsivo e com suporte a Dark Mode.
- **VoIP Integrado**: Ligações integradas ao histórico do cliente.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)

## 📦 Começando

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para build de produção:
   ```bash
   npm run build
   ```
